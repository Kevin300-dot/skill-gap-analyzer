# SkillBridge - Skill Gap Analyzer

A web app I built to help students in the automotive and EV field figure out what skills they're missing and what courses they should take to get hired.

## What it does

You enter your current course, pick a job role you want, and select the skills you already have. The app then shows you exactly what skills you're missing, gives you a readiness score, and recommends specific courses to close the gap.

## Why I built this

I noticed that a lot of engineering and ITI students graduate without knowing what specific skills companies actually want. This app tries to bridge that gap.

## Tech used

- Python
- Streamlit
- Plotly

## Features

- Readiness score shown as a gauge chart
- Pie chart showing your skill coverage
- Course recommendations sorted by priority
- Covers EV Technician, Automobile Manufacturing, Auto Components and Battery Assembly roles

## How to run it

Install the requirements:
pip install streamlit plotly

Then run:
streamlit run app.py
